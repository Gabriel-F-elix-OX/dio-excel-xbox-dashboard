# dio-excel-xbox-dashboard
Projeto DIO para prática de criação de Dashboard

Este Dashboard foi feito para o curso do Excel 365

Por ter sido feito no Excel 365 de Web (versão gratuita) pode ocorrer alguns BUGs, por isso foi feita uma versão no Google Sheets para visualização sem BUGs, o link se consta abaixo

O projeto difere do que foi apresentado na plataforma DIO por problemas de diferenciação da versão do Excel utilizado pelo Professor em relação ao 365 para Web, foi necessária adaptações para manter ele funcional

Encontramos problemas tanto no Sheets quanto no Excel Web de puxar as informações da tabela dinâmica da Aba Cálculos para os gráficos na aba Dashboard, por isso inseri as tabelas dinâmicas na linha 900 da aba dashboard
